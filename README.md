# React Redux Starter Kit

### Deprecated
Please use https://reactjs.org/community/starter-kits.html

### Getting Started

```
> git clone https://github.com/usunyu/react-redux-starter-kit

> cd react-redux-starter-kit

> npm install

> npm start
```

### Getting Ready for Deploy

Beta Testing
```
> npm run beta
```

Production
```
> npm run prod
```
